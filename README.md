# 🛡️ AegisAI — Explainable Self-Healing Data Center Copilot

> *Predict failures. Explain causes. Recommend fixes. Simulate recovery — before downtime happens.*

---

## Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Place your raw SMD file in data/
#    e.g.  data/machine-1-1.txt

# 3. Convert to AegisAI CSV
python convert.py --input data/machine-1-1.txt --output data/smd_sample.csv

# 4. Run the dashboard
streamlit run app.py
```

---

## Dataset Format — SMD (Server Machine Dataset)

The raw input is a headerless `.txt` file with **38 comma-separated float columns**,
all pre-normalised to **[0, 1]** by the original dataset authors.

`convert.py` maps these columns:

| Column index | CSV name    | Unit after convert |
|---|---|---|
| 0  | `cpu`        | % (×100)   |
| 1  | `net_in`     | [0,1] raw  |
| 2  | `net_out`    | [0,1] raw  |
| 3  | `network`    | % (×100)   |
| 4  | `temp`       | °C (30–100)|
| 5  | `power`      | % (×100)   |
| 6  | `mem`        | % (×100)   |
| 7  | `disk_read`  | [0,1] raw  |
| 8  | `disk_write` | [0,1] raw  |
| 9–11 | `load_1/5/15` | [0,1] raw |
| 12–37 | `feat_12`…`feat_37` | [0,1] raw |

A synthetic `timestamp` column (1-minute intervals from 2024-01-01) is added automatically.

## Project Structure

```
AegisAI/
├── app.py                    # Streamlit dashboard
├── convert.py                # SMD .txt → CSV converter
├── requirements.txt
├── data/
│   ├── machine-1-1.txt       # Raw SMD input
│   └── smd_sample.csv        # Generated by convert.py
└── utils/
    ├── __init__.py
    ├── preprocess.py         # Load, clean, feature engineering
    ├── train.py              # IsolationForest + RandomForest
    ├── explain.py            # Natural-language explanations
    ├── recommend.py          # Prioritised remediation playbook
    └── simulate.py           # Counterfactual fix simulation
```

                  --output data/smd_sample.csv    # destination CSV
                  --rows   5000                   # optional: first N rows only
